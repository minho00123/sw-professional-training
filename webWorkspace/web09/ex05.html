인라인 방식

ex h1 style="color:red"

css 문서정의

head의 style태그를 통해 정의

ex head->style->선택자{속성명:속성값;}

css 파일정의

css파일을 만들고 불러오는 방법

ex head->link->href속성으로 파일 로딩

선택자

동일 선택자의 경우는 가장 마지막 적용한 속성 적용

우선순위: 인라인 > 아이디 선택자 > 클래스 선택자 > 태그 선택자

아이디 선택자

문서에 유일한 하나의 아이디를 갖는 엘리먼트에 적용

클래스 선택자

문서의 동일한 클래스에 모두 적용

태그 선택자

문서의 모든 태그에 적용
